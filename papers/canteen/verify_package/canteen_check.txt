摘 要
我国老龄化程度近年来持续加深，社区养老服务的需求量也在增长。老

年食堂是社区养老服务体系的一个组成部分，主要职能是为社区老年居民提

供日常餐饮保障。多数社区老年食堂目前仍用纸质登记和人工统计的办法来

处理订餐业务，菜品信息更新不及时、订单数据汇总困难、用户信息分散在

不同登记本上，这几个问题比较突出。如果把信息化手段引入老年食堂的日

常运营，人工操作环节可以减少，订餐流程的准确性和效率也能有所提升。
本文设计并实现了一套社区老年食堂订餐系统，后端选用 Java 语言和

Spring Boot 框架，数据持久层使用 MyBatis-Plus 操作 MySQL 数据库，前端

管理界面基于 Vue 2 和 Element UI 搭建，用户端页面采用 HTML、CSS 和

JavaScript 编写。系统按管理员、员工、用户三种角色划分功能权限，管理

员负责菜品维护、用户管理、订单处理、供应商管理和公告发布等后台操作，

员工负责菜品上下架和订单发货，用户则在前台完成菜品浏览、在线订餐、

余额充值、积分累计、订单评价和论坛交流等操作。

Abstract
With the aging population in China continuing to grow, community elderly
canteens need information systems to improve the efficiency of meal ordering and
daily operations. This thesis designs and implements a community elderly canteen
ordering system. The backend is developed using Java with Spring Boot
framework, and MyBatis-Plus serves as the data persistence layer operating on
MySQL database. The frontend management interface is built with Vue 2 and
Element UI, while the user-facing pages are developed using HTML, CSS, and
JavaScript. The system supports three user roles: administrators, employees, and
regular users, covering functional modules including dish management, online
ordering, order processing, supplier management, forum interaction, and
announcement publishing.
Functional testing and performance testing verify that all modules operate as
expected, with page response times and database operations within acceptable
ranges, meeting the basic management needs of daily meal ordering in community
elderly canteens.
Key words Community Elderly Canteen; Ordering System; Spring Boot;
Vue; MyBatis-Plus
